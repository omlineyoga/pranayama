# 🧘 OMLINE · Guía Interactiva de Pranayamas

**Bienestar Consciente** — Guía animada de técnicas de respiración del yoga.

🌐 **Ver en vivo:** [https://TU-USUARIO.github.io/omline-pranayamas](https://TU-USUARIO.github.io/omline-pranayamas)

## ¿Qué incluye?

Animación interactiva con **7 técnicas de pranayama**:

| Técnica | Principiante | Completo |
|---|---|---|
| Sama Vritti (Cuadrada) | 3·3·3·3 | 6·6·6·6 |
| Ujjayi (Oceánica) | 3·6 | 4·8 |
| Chandra Bhedana (Lunar) | 3·6 | 4·4·8 |
| Surya Bhedana (Solar) | 3·6 | 4·4·8 |
| Nadi Shodhana | 3·6·3·6 | 4·8·8·4 |
| 4 · 7 · 8 | 4·8 | 4·7·8 |
| Bhramari (Abeja) | 3·6 | 4·8 |

### Características
- ✨ Geometría sagrada animada (Semilla de la Vida, Shatkona)
- 🎵 Sonido de respiración con Web Audio API
- 🏔️ Paisaje zen de fondo
- 🌙 Modo claro y oscuro automático
- 📱 Responsive para móvil y escritorio
